# Skill Creator（摘要版）

> 完整内容: `SKILL_FULL.md`（同目录）

## 核心原则
- **Skills = 模块化的专业能力包**，把通用agent变成领域专家
- **SKILL.md** 是入口，提供必要的上下文和流程
- 技能文件可以是纯Markdown，也可以附带脚本、资源
- **质量优先于数量** — 按需创建，不每事一技能。周期性归纳优于即时创建
- **先查后建** — 创建新技能前先检查是否与现有技能重复，优先扩展现有技能

## 创建流程
1. **确定触发条件** — 在 description 中写明何时触发
2. **写核心指令** — 告诉agent该做什么、怎么做
3. **提供上下文** — 业务流程、检查清单、模板代码
4. **附加资源** — 脚本、配置文件、参考文件

## 好技能的标准
| 维度 | 要求 |
|------|------|
| 专注 | 一个技能只做一件事 |
| 清晰 | 指令明确无歧义 |
| 安全 | 不越过权限边界 |
| 可维护 | 结构清晰易更新 |
