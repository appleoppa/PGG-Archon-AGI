# Skill Creator — Compact

## Trigger

Use when creating a new Hermes skill from a successful workflow or user-requested procedure.

## Workflow

1. Confirm skill has recurring value.
2. Write compact frontmatter, trigger, workflow, pitfalls and verification.
3. Put long examples in references.
4. Validate skill loads.
