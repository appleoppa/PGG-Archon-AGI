## 三、delegate_task调用示例

### 新建民事案件（标准流程）

```python
# Step 1: 申请编号
case_id = delegate_task(
    goal="申请案件编号：\n案件类型：民事\n当事人：XXX（原告）/YYY（被告）\n案由：建设工程施工合同纠纷",
    context="",
    skills=["agent-cms"],
    toolsets=["terminal", "file", "web"]
)

# Step 2: 派发至民事案件部
dept_result = delegate_task(
    goal=f"代理原告处理案件 {case_id}：\n案情摘要：...（详见案件材料）",
    context="案件目录：workspace/案件共享/{case_id}/",
    skills=["civil-litigation"],
    toolsets=["terminal", "file", "web"]
)

# Step 3: 巡视组效验（并行）
xunshi_result = delegate_task(
    goal=f"巡视案件 {case_id} 节点①：\n请执行分析报告草案效验",
    context="",
    skills=["inspection-team"],
    toolsets=["terminal", "file", "web"]
)

# Step 4: 审计组审查
shenji_result = delegate_task(
    goal=f"审计案件 {case_id}：\n执行10项审计清单",
    context="",
    skills=["audit-team"],
    toolsets=["terminal", "file", "web"]
)
```

### 完整案件流程示例
1. 用户提出法律需求
2. 苹果中枢判断：快问（轻桥层）→ 直接回答 / 标准流程 → 申请编号
3. an-guan分配PGG编号，创建办案库
4. 派发至对应业务部门（min-shi/xing-shi等）
5. law部门提供法律依据支持
6. xun-shi巡视组在关键节点效验
8. zhi-nao智脑知识部知识沉淀
9. shen-ji审计组最终审查
10. 聚合交付，通知用户
