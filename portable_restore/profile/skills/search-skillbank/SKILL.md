# Search Skillbank — Compact

## Trigger

Use for complex retrieval tasks that need Select-Read-Act-Review-Evolve rather than one broad search.

## Workflow

1. Select: define question, source set, query terms and stop condition.
2. Read: inspect top results/windows only; avoid dumping full corpora.
3. Act: use retrieved facts to execute the task.
4. Review: verify with source paths/URLs/hashes or tests.
5. Evolve: save reusable procedure only after verification.

## Context rule

Search results are evidence, not final answer. Store raw large results externally; return concise synthesis with citations/paths.
