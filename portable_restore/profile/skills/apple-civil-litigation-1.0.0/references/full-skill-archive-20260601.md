# 苹果民事诉讼专家 V1.1.0

> 民商事诉讼全流程法律护航 | 合同纠纷 · 房产纠纷 · 建设工程 · 民间借贷

## 🎯 核心能力

- ⚖️ **全流程诉讼** - 从立案到执行的完整诉讼指导
- 🤖 **MiniMax大模型** - 深度推理法律问题与诉讼策略
- 📚 **专业精准** - 房地产领域民商事纠纷深度覆盖
- 📝 **文书起草** - 起诉状/答辩状/代理词/上诉状

## 📋 适用场景

- 死亡赔偿金/精神抚慰金内部共有物分割纠纷（见参考文件）
- 重复起诉抗辩分析（交通事故外部求偿 vs 内部分配的关系）
- 商品房买卖合同纠纷（逾期交房/逾期办证/质量瑕疵）
- 建设工程施工合同纠纷（工程款/工期/质量）
- 房屋租赁合同纠纷（租金/腾退/装修损失）
- 民间借贷纠纷（本息计算/担保/夫妻共债）
- 股权转让纠纷/合作开发纠纷
- 仲裁程序指导
- **离婚纠纷（含彩礼返还）**（见参考文件）

## 📋 报告质量标准（强制）

所有分析报告必须包含以下要素：

### 🔴 事实溯源声明（必备）

报告中涉及当事人的**每条事实断言**必须标注来源：

| 事实断言 | 来源 | 可信度 |
|---------|------|--------|
| 示例：持刀划脖子 | 📄 苹果哥原始输入 / 📷 聊天截图OCR（文件：行号） | ✅ 确认 / ⚠️ 推测 |

**来源类型**：
- `📄 苹果哥原始输入` — 用户口述/说明文档
- `📷 聊天截图OCR` — OCR提取的截图文字（附文件名和行号）
- `📝 当事人自述` — 客户直接提供
- `⚖️ 合理推导` — 未直接确认的推理（必须标注推测）

**禁止**：凭空添加无来源的事实断言（详见 apple-inspection-team 巡视规则）

### 📐 报告输出规范
- 完成状态标注：草稿 / 待复核 / 已复核
- 法条引用：精确到款、项，版本标示（如法释〔2024〕1号）
- 不确定事项用 ⚠️ 标注

## 🔧 核心功能模块

### 1. 诉前准备
- 诉讼请求确定（具体金额/违约金/利息计算）

- 证据收集与整理（证据清单编制）

### 3. 二审程序
- 上诉状起草（上诉请求与理由）

### 4. 执行程序
- 执行申请起草
- 财产查控申请
- 追加/变更被执行人
- 执行异议/复议
- 参与分配申请

### 5. 仲裁程序
- 仲裁协议效力确认
- 仲裁申请起草
- 仲裁反申请
- 仲裁裁决撤销/不予执行

## 🏠 房地产专题

### 商品房买卖合同纠纷
- 逾期交房：违约金计算标准（司法解释第13条）
- 逾期办证：违约金起算点与诉讼时效
- 房屋质量问题：保修责任 vs 违约责任 vs 侵权责任
- 一房二卖：房屋归属与惩罚性赔偿
- 限购/限贷：定金退还与缔约过失责任

### 建设工程施工合同纠纷
- 工程款结算（固定价 vs 成本加酬金）
- 工期延误责任（发包人原因 vs 承包人原因）
- 质量异议（验收程序/保修责任）
- 实际施工人突破合同相对性（建工司法解释一第43条）
- 建设工程价款优先受偿权

### 房屋租赁纠纷
- 租金标准调整（疫情/市场变化）
- 优先购买权（通知义务/同等条件）
- 装饰装修处置（附和/拆除）
- 腾退纠纷（公告送达/执行难点）

## 💔 离婚纠纷专题（含彩礼返还）

### 核心法律框架

| 请求 | 法条依据 | 适用条件 |
|------|---------|---------|
| **判决离婚** | 《民法典》第1079条第三款 | (二)家庭暴力 / (四)分居满2年 / (五)其他致感情破裂 |
| **撤销婚姻** | 《民法典》第1053条 | 一方隐瞒重大疾病（如抑郁症），另一方可1年内请求撤销 |
| **彩礼返还** | 民法典婚姻家庭编解释一第5条 + 2024彩礼新规（法释〔2024〕1号） | 短婚+无子女+大额+给付致生活困难+女方过错 |
| **损害赔偿** | 《民法典》第1091条 | 家暴/遗弃/重大过错致离婚 |

### 常见彩礼情形分析表

| 情形 | 法律效果 | 返还概率 |
|------|---------|---------|
| 未登记结婚 | 原则上全额返还 | 80%~100% |
| 已登记+未共同生活 | 解释一第5条(二)，离婚为条件要求返还 | 60%~90% |
| **已登记+共同生活<1年** | **2024新规核心情形，考虑短婚+无子女+大额** | **部分返还20%~45%** |
| 已登记+共同生活>1年 | 原则上不返还（特殊情形例外） | <10% |

### 证据收集要点
- **彩礼给付凭证**：转账记录/取款回单/发票——**绝大多数缺失，先查**
- **暴力行为证据**：报警记录/伤情照片/聊天记录——证明家庭暴力，影响离婚判决
- **隐瞒病情证据**：婚前确诊记录/女方自认聊天记录——可走撤销婚姻路径
- **生活困难证明**：村委会证明/举债证据/借款借条——彩礼返还的核心要件
- **聊天记录导出**：完整导出原始聊天记录（非仅截图），关键词搜索定位

### 🔧 截图证据OCR提取法（当模型不支持视觉时）

```
tesseract /path/to/截图.jpg /tmp/output.txt -l chi_sim
```
1. 确认 tesseract 已安装且 chi_sim 中文字库就绪
2. 对每张截图执行 OCR
3. 读取输出文件（tesseract 自动加 .txt 后缀：output.txt.txt）
4. 整理关键信息，标注文件名+行号作为溯源依据

### 诉讼策略路径（A/B双轨）
- **路径A（诉讼）**：离婚诉讼主攻家暴（1079条(二)）+ 彩礼返还按2024新规（短婚+共同生活短+大额+女方过错）
- **路径B（调解）**：分三级谈判阶梯，底线约彩礼总额20~25%

## 📊 费用计算

### 诉讼费用速算
- 财产案件：按诉讼标的额分段计算（1.2万以下=66元；1.2万~9.6万=240元+超出部分×2%；9.6万~96万=1920元+超出部分×1.5%...）
- 保全费：不超过5000元部分按1%计算，最低不少于30元
- 执行费：按执行到位金额计算（无休止金先扣后退）

### 律师费参考
- 各地司法局/律师协会有收费标准（仅供参考）
- 风险代理一般不超过标的额的30%
