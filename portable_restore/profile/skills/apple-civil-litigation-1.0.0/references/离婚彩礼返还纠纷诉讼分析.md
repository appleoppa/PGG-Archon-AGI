# 离婚彩礼返还纠纷诉讼分析框架

> 本文件记录离婚纠纷+彩礼返还案的标准分析流程。基于2026-05-22真实案件（017号）执行沉淀。

## 一、标准办案流程

```
接案（确认代理方）
  → 案件管理中心编号（agent-cms）
  → 并行派发：民事案件部 + 证据管理部 + 诉讼策略部
    ├── analysis-法律分析报告.md
    ├── analysis-证据管理报告.md
    └── analysis-诉讼策略报告.md
  → 巡视组复核（inspection-巡视报告.md）
  → 整改闭环（事实溯源修复）
  → 综合交付（交付-综合报告.md）
  → 归档
```

## 二、核心法律路径

### 2.1 离婚路径（三选一或并行）

| 路径 | 法条 | 适用条件 | 优势 | 风险 |
|------|------|---------|------|------|
| **A. 家庭暴力** | 1079条(二) | 持刀/暴力等行为，需证据 | 法定离婚事由 | 证据不足时难认定 |
| **B. 隐瞒重大疾病** | 1053条 | 抑郁症等严重疾病婚前未告知 | 可撤销婚姻→彩礼全退 | 抑郁症是否属"重大疾病"有争议 |
| **C. 兜底条款** | 1079条(五) | 综合因素致感情破裂 | 适用范围广 | 法官自由裁量大 |

### 2.2 彩礼返还的2024新规（法释〔2024〕1号）

**核心考量因素**（逐项标注证据状态）：
1. 共同生活时间长短（≤1年 → 有利）
2. 彩礼数额大小（30万+ → 大额 → 有利）
3. 是否孕育子女（无 → 有利）
4. 双方过错（女方暴力/辱骂 → 有利）
5. 给付方经济状况（举债给付 → 有利）
6. 当地风俗习惯

> ⚠️ 传统解释一第5条第(三)项"给付致生活困难"仍是可选路径，但2024新规提供了更灵活的裁量框架。

## 三、事实溯源标准（巡视强制要求）

每条事实断言必须标注来源。标准格式（用于报告头部）：

```
## 🔴 事实溯源声明

| 事实断言 | 来源 | 可信度 |
|---------|------|--------|
| 结婚刚满一年 | 📄 部分说明.md | ✅ 确认 |
| 彩礼订婚礼8.8万 | 📷 聊天截图OCR（聊天2第18-19行） | ✅ 确认 |
| 女方持刀划脖子 | 📄 部分说明.md + 📷 聊天截图OCR | ✅ 确认 |
| 彩礼合计30万+ | ⚠️ 推测：8.8万+6~7万+其他估算 | ⚠️ 估算 |
```

**来源标识规范**：
- `📄 文件来源` — 用户提供的文档/说明
- `📷 截图OCR` — OCR提取的截图文字（附文件名+行号）
- `📝 客户自述` — 客户直接提供
- `⚖️ 合理推导` — 未直接确认的推理（必须标注推测）

## 四、证据要点速查

### 4.1 必须优先收集（48小时内）

| 优先级 | 证据类型 | 获取方式 |
|--------|---------|---------|
| 🔴 P0 | 完整聊天记录导出 | 微信→我→设置→聊天→聊天记录迁移与备份→导出 |
| 🔴 P0 | 彩礼支付凭证 | 银行/微信/支付宝转账记录、取款回单 |
| 🔴 P0 | 报警记录调取 | 事发地派出所（如有报警） |
| 🟡 P1 | 伤情照片/医院记录 | 如有划伤/就诊记录 |
| 🟡 P1 | 女方抑郁诊断记录 | 陪同就医病历、医院记录 |
| 🟢 P2 | 村委会贫困证明 | 证明给付彩礼导致生活困难 |
| 🟢 P2 | 借款凭证/借条 | 凑彩礼的借款记录 |
| 🟢 P2 | 媒人/亲属证言 | 证明彩礼交付事实 |

### 4.2 OCR证据提取法

当模型不支持图像视觉时，用 tesseract 提取：

```bash
# 单张截图
tesseract /path/to/截图.jpg /tmp/output.txt -l chi_sim

# 批量
for f in /path/*.jpg; do
    tesseract "$f" "/tmp/$(basename "$f" .jpg)" -l chi_sim
done

# 输出文件为 /tmp/截图.txt.txt (tesseract自动加.txt后缀)
```

**注意**：OCR 结果有噪声（中文字符识别不完美），需人工校对关键数字和人名。

## 五、诉讼策略模板

### 诉讼请求（主策略）
```
1. 判令原告与被告离婚；
2. 判令被告返还原告彩礼人民币____元（或返还比例）；
3. 本案诉讼费用由被告承担。
```

### 备选请求
```
4. 判令被告赔偿原告因家庭暴力造成的精神损害赔偿人民币____元；
5. 判令被告返还原告购赠的三金首饰/折价赔偿。
```

### 调解策略（三级阶梯）
| 阶梯 | 目标金额 | 策略 |
|------|---------|------|
| 第一轮 | 15~20万 | 摆证据、亮底线（诉讼成本/家暴认定/时间成本） |
| 第二轮 | 10~15万 | 强调短婚+隐瞒抑郁+家暴过错 |
| 底线 | 8万（~25%） | 退无可退，低于此数额诉讼 |

## 六、类案检索关键词

- 彩礼返还 + 短婚 + 一年以内
- 婚前隐瞒抑郁症 + 离婚 + 彩礼
- 家庭暴力 + 离婚 + 彩礼返还
- 民法典第1053条 + 重大疾病 + 撤销婚姻
- 法释〔2024〕1号 + 彩礼纠纷
- 给付导致生活困难 + 彩礼返还
