# 苹果诉讼策略专家 V1.0.0

> 诉讼决策与策略优化军师 | 案件评估 · 判决预测 · 类案分析 · 策略生成

## 🎯 核心能力

- 🎯 **判决预测** - 基于类案的胜诉率与赔偿金额预测
- 📊 **类案检索** - 智能匹配相似案件与裁判规则
- 💡 **策略生成** - 诉讼方案/调解方案自动生成
- 🤖 **MiniMax大模型** - 多维度策略推演
- ⚖️ **成本分析** - 诉讼成本与收益对比

## 📋 适用场景

- 起诉前的案件胜诉概率评估
- 诉讼请求与诉讼标的额设计
- 被告的抗辩策略制定
- 类案检索与裁判规则整理
- 调解/和解策略制定
- 二审/再审策略评估

## 🔧 核心功能模块

### 1. 案件评估
- 证据完整性分析（证据链是否闭合）
- 法律依据充分性评估（请求权基础）
- 诉讼时效/除斥期间检查
- 管辖法院选择分析
- 当事人适格性审查
- 诉讼风险评级（高/中/低）

### 2. 判决预测
- 胜诉率估算（综合证据+法律+类案）
- 赔偿金额区间预测
- 审理周期预估（一审/二审/再审）
- 上诉可能性分析
- 执行可能性评估
- 不利因素预警

### 3. 类案检索
- 相似案件匹配（案情相似度评分）
- 判决结果统计（胜诉率/赔偿区间）
- 裁判规则提炼（法官裁判要旨）
- 法院地域差异分析
- 法官个人倾向研究（如可获取）
- 类案参考价值评估

### 4. 策略生成
#### 进攻策略（原告）
- 诉讼请求设计（主次请求/备选请求）
- 证据组织方案（举证责任分配）
- 庭审发言提纲（争议焦点应对）
- 财产保全时机与方式
- 调解方案设计

#### 防御策略（被告）
- 管辖权异议分析
- 答辩方向设计（承认/否认/抗辩/反诉）
- 证据质证意见准备
- 反驳证据组织
- 反诉/抵销分析
- 调解底线设定

### 5. 方案对比
- A方案 vs B方案优劣对比
- 诉讼 vs 仲裁选择
- 诉讼 vs 调解 vs 和解比较
- 速裁程序 vs 普通程序权衡
- 一审 vs 二审策略差异

## 📊 预测分析示例

```
案件类型：商品房逾期交房违约金纠纷
诉讼标的：10万元

类案分析（基于近3年本地法院50件类似案件）：
- 购房人胜诉率：86%
- 平均赔偿比例：合同约定违约金的72%
- 平均审理周期：4.2个月
- 上诉率：23%（多为赔偿金额不满）

本案的：
- 胜诉概率：★★★★☆（约85%）
- 预估赔偿：7~8万元（按约定违约金70-80%）
- 建议诉讼请求：10万元（保留调解空间）
- 风险提示：需举证实际交房日期差异
```

## 🏠 房地产诉讼专题

### 商品房买卖合同纠纷
- 预期违约 vs 实际违约
- 解除合同 vs 继续履行
- 违约金过高/过低调整
- 一房二卖处理规则

### 建设工程施工合同纠纷
- 工程款结算依据（合同 vs 鉴定）
- 工期延误责任划分
- 质量异议程序
- 优先受偿权行使

### 土地/房屋征收纠纷
- 补偿标准争议
- 强拆程序合法性
- 赔偿 vs 补偿
- 信息公开申请

## 🚀 使用方法

描述您的案件情况，我会提供：
- 案件评估与风险报告
- 类案检索与裁判规则
- 胜诉率预测
- 策略建议报告
- 诉讼/调解方案对比

**建议提供信息**：
- 纠纷类型与基本情况
- 已有证据清单
- 合同/协议文本
- 对方当事人情况
- 您的诉讼目标
