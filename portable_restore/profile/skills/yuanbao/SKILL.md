# Yuanbao — Compact

## Trigger

Use for Yuanbao group/direct messaging, @mentions, member lookup and message delivery.

## Workflow

1. Identify target account/group and whether action is read/send.
2. For specific person/channel, list/resolve targets first if needed.
3. Compose concise message and include media paths when requested.
4. Send via Yuanbao tool and report delivery evidence.

## Boundary

External messages are side effects; require clear user intent.
