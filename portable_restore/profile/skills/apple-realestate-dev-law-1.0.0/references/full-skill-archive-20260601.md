## 📊 行业特殊规定

### 商品房预售条件

| 条件 | 要求 | 说明 |
|-----|------|------|
| 土地使用权 | 已取得土地使用权证书 | 无抵押或已解押 |
| 建设工程规划 | 已取得建设工程规划许可证 | 符合规划条件 |
| 施工许可 | 已取得建筑工程施工许可证 | 可以合法开工 |
| 资金投入 | 已完成投资≥25% | 证明资金实力 |
| 施工进度 | 已确定施工进度和竣工日期 | 可以按期交付 |

### 房地产调控政策

**限购政策**（因城施策）：
- 户籍限制
- 社保/个税年限要求
- 已有住房套数限制
- 离婚购房限制

**限贷政策**：
- 首套房首付比例（20-30%）
- 二套房首付比例（40-80%）
- 贷款利率（LPR+基点）
- 三套房停贷

**限售政策**：
- 取得产权证满2/3/5年方可转让
- 打击投机炒房

**价格管控**：
- 预售价格备案
- 涨价幅度限制
- 一房一价公示

### 工程款优先受偿权

**行使条件**：
- 发包人未按约定支付工程款
- 催告后合理期限内仍未支付
- 建设工程质量合格

**优先范围**：
- 工作人员报酬
- 材料款等实际支出
- 不包括发包人违约损失

**行使期限**：
- 自发包人应当给付工程款之日起18个月

**优先顺位**：
- 优于抵押权和其他债权
- 但劣后于消费者购房款
