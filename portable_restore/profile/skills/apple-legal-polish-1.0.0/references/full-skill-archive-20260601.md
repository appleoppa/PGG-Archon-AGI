### 合同批注式审阅
```
审阅本借款合同，以批注的形式进行修改，并且输出至桌面
```

**技术流程：**

1. **读取docx**：使用 `zipfile + re` 去除XML标签提取纯文本，或用 `pandoc` 转换
2. **定位条款**：遍历 `doc.paragraphs`，用 `target in para.text` 匹配原文本片段
3. **添加批注**：用 `para.add_run()` 添加红色小字批注
   ```python
   run = para.add_run()
   run.text = "\n\n【AI批注】" + comment
   run.font.color.rgb = RGBColor(0xC0, 0x00, 0x00)  # 红色
   run.font.size = Pt(9)
   ```
4. **输出路径**：保存至桌面，原文件名 + "_批注版"

**常见法律问题清单（借款合同）：**
- 借款金额：大写与实际到账金额的关系、转账凭证出具时间
- 借款期限：起算日固定、年利率上限（民法典LPR4倍）、展期条件具体化
- 还款方式：还款日期、本金利息分配、逾期利息、还本付息明细表
- 资金监管：出借人对资金流向的监控权
- 担保条款：类型、人信息、期限、实现债权条件（避免"担保（如有）"）
- 违约责任：逾期利息标准、提前还款违约金、律师费承担
- 争议管辖：仲裁前置、被告所在地或合同履行地
- 签署页：签署日期、签约地点、骑缝章

**输出格式：** 中文汇报，简洁直接，批注内容分点列举，不展示技术过程。
