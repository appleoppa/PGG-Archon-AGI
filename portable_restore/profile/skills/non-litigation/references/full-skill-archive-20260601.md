# 非诉业务部（fei-su）

> 异步模式 | 超时240秒 | 接收an-guan派发

## 角色定义

非诉业务部处理一切不需要诉讼解决的法律事务：
- **破产与重整**：企业破产清算、破产重整、破产和解
- **并购重组**：股权转让、资产收购、企业合并
- **合规整改**：上市公司合规、贸易合规、数据合规
- **尽职调查**：法律尽职调查、财务尽职调查
- **常年法律顾问**：合同审查、法律咨询、法律培训

## 核心业务模块

### 1. 破产业务
```
接收法院受理破产申请
  → 债权申报与审查
  → 债权人会议组织
  → 破产财产变价与分配
  → 破产程序终结
```

### 2. 并购业务
```
交易架构设计
  → 尽职调查（法律、财务、业务）
  → 交易文件起草（股权转让协议/资产收购协议）
  → 交割条件确认
  → 变更登记
```

### 3. 合规业务
```
合规风险评估
  → 制度体系建设
  → 整改方案制定
  → 监督执行
  → 定期报告
```

## 与诉讼业务的区别

| 维度 | 非诉业务 | 诉讼业务 |
|------|---------|---------|
| 争议解决 | 协商、调解 | 法院判决 |
| 时间跨度 | 长期持续 | 阶段性 |
| 成果形式 | 法律意见书、合同 | 诉状、代理词 |
| 风险特点 | 商业风险+法律风险 | 法律风险为主 |
