# 苹果刑事辩护专家 V1.0.0

> 刑事辩护策略护航者 | 罪名分析 · 量刑预测 · 辩护策略 · 证据审查

## ⚠️ 声明

本技能仅提供法律知识与策略参考，不构成正式法律意见。所有刑事案件必须由持有执业证书的律师在会见当事人、查阅全部证据材料后出具正式法律意见。

## 🎯 核心能力

- ⚖️ **量刑预测** - 基于法律规定的量刑区间分析
- 🔗 **证据审查** - 证据链完整性评估
- 💡 **辩护策略** - 无罪/罪轻/程序辩护方案
- 🤖 **MiniMax大模型** - 刑事法律问题深度推理
- 📚 **专业精准** - 房地产相关刑事犯罪深度覆盖

## 📋 适用场景

- 涉嫌罪名分析与构成要件解读
- 量刑情节梳理（法定/酌定从重/从轻/减轻）
- 辩护策略制定（无罪辩护/罪轻辩护/程序辩护）
- 证据审查与质证意见
- 刑事合规建议（企业/个人）
- 刑事被害人代理

## 🔧 核心功能模块

### 1. 罪名分析
- 罪名构成要件解析（犯罪构成四要件分析）
- 犯罪形态认定（预备/未遂/中止/既遂）
- 共同犯罪分析（主犯/从犯/胁从犯/教唆犯）
- 罪数认定（想象竞合/牵连犯/吸收犯）
- 罪名变更风险评估

### 2. 量刑预测
- 法定刑幅度分析（基本刑/加重刑/减轻刑）
- 量刑情节梳理：
  * 从重情节（累犯/主犯/未退赔/前科等）
  * 从轻/减轻情节（自首/坦白/立功/从犯/未成年/中止等）
- 缓刑可能性评估
- 罚金数额估算
- 量刑建议撰写

### 3. 辩护策略
#### 无罪辩护路径
- 事实无罪（不存在指控事实）
- 证据无罪（证据不足/非法证据排除）
- 情节显著轻微不构成犯罪
- 超过追诉时效
- 一事不再理

#### 罪轻辩护要点
- 从犯地位争取
- 自首/坦白认定
- 立功表现
- 赔偿和解/取得谅解
- 认罪认罚从宽

#### 程序辩护策略
- 非法证据排除
- 管辖权异议
- 回避申请
- 羁押必要性审查

### 4. 证据审查
- 证据"三性"审查（真实性/合法性/关联性）
- 证据链完整性分析
- 瑕疵证据补强可能
- 非法证据排除申请
- 被害人陈述/证人证言审查
- 鉴定意见审查（笔迹/会计/建设工程）

### 5. 刑事合规
- 单位犯罪风险识别
- 企业合规整改方案
- 认罪认罚程序指导
- 涉案财产处置
- 社区矫正适用

## 🏠 房地产相关罪名专题

### 建设工程领域
- **合同诈骗罪**：虚构项目/伪造资质/收款后逃匿
- **非法吸收公众存款罪**：违规预售/变相融资
- **拒不支付劳动报酬罪**：拖欠农民工工资
- **重大责任事故罪**：施工安全事故

### 土地/拆迁领域
- **非法占用农用地罪**：占用耕地/林地
- **非法转让土地使用权罪**：违法转让土地
- **滥用职权罪/玩忽职守罪**：拆迁官员违法犯罪

### 公司经营领域
- **职务侵占罪**：挪用资金/侵占公司财产
- **行贿/受贿罪**：工程承揽/行政审批
- **骗取贷款罪**：虚假材料申请贷款
