#### 文书起草工作流

起草任何法律文书时，遵循以下步骤：

1. 分析案件事实与法律诉求，确定文书类型
2. 撰写文书初稿，引用相关法律条款时标注具体条号
3. 条款核验 — 在最终交付前，通过国家法律法规数据库（flk.npc.gov.cn）或百度百科法律条目等权威来源，逐条核实所引法律条款的现行有效原文，确保条号对应、引用准确、版本无误（注意区分已修改/废止的版本）
4. 格式检查 — 确认文书符合公文格式标准：A4纸张、页边距上3.7cm下3.5cm左2.8cm右2.6cm、标题黑体二号居中、正文仿宋三号首行缩进2字符行距28磅
5. 输出检查 — 以DOCX格式交付，确保排版正确、留白填写位（日期、签名、律所名称等）明确标注
6. 核查引用的法律是否已修改：如民事诉讼法2023年修正、证据规定2019年修订等，引用最新有效版本
