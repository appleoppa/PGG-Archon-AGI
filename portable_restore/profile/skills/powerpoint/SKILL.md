# PowerPoint — Compact

## Trigger

Use to create, read, edit, split, merge, or inspect PPTX files and slide decks.

## Workflow

1. Confirm goal: outline, design, edit, extract, convert, or audit.
2. Inspect existing deck structure before editing.
3. Preserve theme/layout when modifying.
4. For generated decks, create clear title, agenda, content slides, and closing/action slide.
5. Verify file exists and can be opened/parsed.

## Output

Return absolute file path, slide count, changes made, and verification result.
