# 2026-05-22 超级进化吸收闭环补齐

## 触发信号

用户指出：“继续完成你没有搞的两步”。本轮暴露的问题不是材料学习不足，而是沉淀闭环不完整：先创建了新 umbrella skill，但没有立即完成已有技能联动和工作区索引报告。

## 可复用教训

当用户要求“全部学习、举一反三、融会贯通、组合最强技能”时，完成标准不只是新建一个 SKILL.md。必须同时完成闭环：

1. **新技能沉淀**：把材料融合成 class-level umbrella skill。
2. **旧技能联动**：检查并 patch 已加载或相关 umbrella skills，避免新技能孤岛化。
3. **报告/索引落地**：在 workspace 生成简洁索引，记录源材料、子智能体分工、融合公式、落地文件和触发方式。
4. **验证**：验证技能可读、相关 patch 可定位、报告存在、源材料完整列入。

## 推荐闭环门禁

交付前自检：

- [ ] 是否只新建了技能，但没有联动相关旧技能？
- [ ] 是否只输出总结，但没有生成可回查索引？
- [ ] 是否把“学习完成”误报为“系统闭环完成”？
- [ ] 是否验证了新技能、旧技能 patch 和报告文件？

## 适用范围

适用于：长上下文学习、外部方法论吸收、超级进化材料整理、技能库升级、开智进化材料吸收。

不适用于：简单问答、单文件小修改、一次性摘要。
