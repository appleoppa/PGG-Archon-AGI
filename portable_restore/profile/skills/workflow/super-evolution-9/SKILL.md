# Super Evolution 9 — EvoMaster Compact

## Trigger

Use for EvoMaster native evolution core tasks: strategy selection, scoring, import, evolution, recommendation, and statistics.

## Core actions

- `select_strategy`: choose evolution strategy from task evidence.
- `score`: compute bounded quality/readiness score.
- `import`: absorb verified pattern.
- `evolve`: apply low-risk improvement.
- `recommend`: propose next action.
- `stats`: report state/readback.

## Truth gates

No evolution claim without implemented artifact plus verification. Keep scores bounded and downgrade when tests fail.
