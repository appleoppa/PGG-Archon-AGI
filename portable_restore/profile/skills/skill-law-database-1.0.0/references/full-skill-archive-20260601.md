# 法律法规库底座（摘要版）

> 完整内容: `SKILL_FULL.md`（同目录）

## 核心定位
九章法律AI帝国的「法典基石」，所有法律技能共享的权威法规数据源。

## 引用规范
- **案例引用**: `applicable_laws: ["LAW-CIVIL-2020-001:第103条"]`
- **全文调取**: `jiuzhang-cli law get LAW-CIVIL-2020-001`
- **版本检查**: `jiuzhang-cli law check --date 2024-01-01`

## 法规管理规则
1. 每个法规唯一编号（领域-年份-序号）
2. 引用时必须标注版本日期
3. 废止/修订/新法自动更新标识
4. 法规与案例自动关联

## 覆盖范围
- 民法、刑法、行政法、经济法
- 民事诉讼法、刑事诉讼法、行政诉讼法
- 司法解释、部门规章、地方法规
- 关键行业法规（房地产、金融、知产等）
