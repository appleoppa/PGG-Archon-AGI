# 018号案例推演示例

> 案件：民邦人力 vs 燕赵财产保险 — 雇主责任保险合同纠纷
> 推演日期：2026-05-17

## 推演结构（可复用模板）

推演报告包含三大模块：

### 一、风险预测

包含关键变量分析表（变量→权重→对胜率的影响）、时间与成本预测表。

### 二、模拟庭审

每个核心争议点的模拟辩论：
- 原告主张 + 被告反驳 + 审判员倾向评估（百分比）
- 形式：争议→双方论点→模拟评估

### 三、策略推演

#### 多路径对比表
| 路径 | 法律依据 | 预期胜率 | 适用条件 |

#### 不同证据情形下的策略调整
- 情形A（被告证据完备）：策略调整
- 情形B（被告证据有瑕疵）：策略调整
- 情形C（被告无有效证据）：策略调整

## 关键原则

1. 推演是分析报告的一部分，推演内容必须整合入终版分析报告
2. 推演报告文件仅存案件档案，不交付
3. 依据复核必须覆盖推演引用的法条
4. 巡视组必须审查分析报告+推演的完整逻辑链
