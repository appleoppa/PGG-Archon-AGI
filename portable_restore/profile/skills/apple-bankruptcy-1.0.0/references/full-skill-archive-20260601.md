# 苹果破产法律专家 V1.0.0

> 房地产开发企业破产法律护航者 | 破产重整 · 和解 · 清算全流程

## 🎯 核心能力

- ⚖️ **破产重整** - 重整计划设计与优化
- 💰 **债权分析** - 债权申报与清偿分析
- 🏢 **企业管理** - 债务人财产管理与处置
- 🤖 **MiniMax大模型** - 破产策略深度推理
- 📚 **专业精准** - 房地产破产领域深度覆盖

## 📋 适用场景

- 房地产企业破产申请与受理
- 破产重整计划设计与可行性评估
- 破产和解协议起草与谈判
- 破产清算财产处置与分配
- 债权人会议策略与权益保护
- 建设工程价款优先权处理

## 🔧 核心功能模块

### 1. 破产申请与受理
- 破产原因分析（不能清偿到期债务/资产不足以清偿全部债务）
- 申请主体资格审查（债权人申请 vs 债务人申请）
- 管辖法院选择（不动产所在地法院）
- 申请材料准备（债务清册、财产状况说明、债权债务清单）

### 2. 重整程序
- 重整可行性评估（经营能力、债务规模、资产价值）
- 重整计划草案设计（债务调整方案、经营优化方案）
- 债权人分类与分组表决（担保债权组、职工债权组、税务债权组、普通债权组）
- 出资人权益调整方案
- 重整计划执行监督

### 3. 和解程序
- 和解协议起草（债务减免、还款期限、担保安排）
- 债权人谈判策略（多数决要求、个别和解禁止）
- 和解条件优化（清偿比例、清偿方式、违约责任）
- 法院认可申请与监督执行

### 4. 清算程序
- 清算组组建与职责
- 财产清理与评估（土地/房产/应收账款/股权投资）
- 债权确认与审查（建设工程优先权、购房人债权、抵押债权）
- 财产变价与分配方案（网络拍卖/评估/协议转让）
- 破产程序终结与注销

### 5. 债权人权益保护
- 债权申报指导与材料准备
- 债权人会议参加策略与表决权行使
- 异议债权处理（向管理人提出/向法院起诉）
- 建设工程价款优先权主张
- 购房人权益保护路径
- 追收债务人财产方案（撤销权/无效行为认定）

## 🏗️ 建设工程特别条款

### 建设工程价款优先受偿权
- 行使期限：建设工程竣工之日或约定竣工之日起18个月内
- 优先范围：实际施工费+材料款+人员费用+违约金
- 优先顺位：在抵押权之前受偿
- 购房人债权与工程款优先权冲突处理

### 房地产开发企业特殊问题
- 土地增值税清算与破产债权
- 预售商品房处置（继续履行 vs 解除）
- 配套公共设施移交问题
- 物业专项维修资金处理

## 📝 典型问题解答

**Q：房地产企业破产中，购房人已支付全款但未交房，如何处理？**
A：购房人可主张取回权（物权期待权）或申报债权，同时注意建设工程价款优先权可能优先于购房人债权。

**Q：建设工程价款优先权与抵押权哪个优先？**
A：建设工程价款优先权 > 抵押权 > 其他普通债权。但需注意购房人已支付大部分房款时，购房人债权可能对抗工程款优先权。

**Q：破产申请被受理后，已开始的诉讼如何处理？**
A：破产申请受理前已进入执行程序的债权可申报债权；正在进行的诉讼/仲裁中止，等候管理人接管。
