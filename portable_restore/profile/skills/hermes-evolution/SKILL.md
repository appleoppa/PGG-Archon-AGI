# Hermes Evolution — Compact

## Trigger

Use to turn user corrections, repeated failures and discovered workflows into AGENTS.md/TOOLS.md/SKILL.md improvements.

## Workflow

1. Identify durable lesson and evidence.
2. Choose target: memory for stable fact, skill for procedure, config/code for executable fix.
3. Patch minimal relevant file.
4. Verify by readback/test.
5. Avoid saving transient task progress.
