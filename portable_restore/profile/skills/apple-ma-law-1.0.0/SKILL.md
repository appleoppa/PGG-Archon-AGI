# Apple M&A Law — Compact

## Trigger

Use for mergers/acquisitions, equity transfer, asset purchase, due diligence, transaction structure, SPA/SHA review, closing and post-closing risk.

## Workflow

1. Identify transaction type, parties, target, consideration, timeline and approvals.
2. Run due diligence checklist: corporate, assets, contracts, debt, labor, tax, litigation, compliance.
3. Map risks to conditions precedent, reps/warranties, indemnity and closing deliverables.
4. Review documents for control, price adjustment, breach and dispute clauses.
