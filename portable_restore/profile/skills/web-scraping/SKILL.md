# Web Scraping — Compact

## Trigger

Use for webpage crawling, data extraction, HTML/API reverse engineering, pagination, anti-bot aware collection and structured export.

## Workflow

1. Check robots/terms and task legality.
2. Prefer official API/static JSON before browser automation.
3. Fetch small sample, inspect selectors/network, then scale.
4. Handle pagination, retries, rate limits and encoding.
5. Save raw/processed data separately and verify row counts/sample fields.

## Boundary

Do not bypass access controls, paywalls, CAPTCHAs or authentication without authorization.
