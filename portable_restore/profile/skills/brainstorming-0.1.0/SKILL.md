# Brainstorming — Compact

## Trigger

Use for creative exploration, idea generation, naming, product/design options and divergent thinking before narrowing.

## Workflow

1. Clarify goal, audience and constraints.
2. Generate diverse options, not one safe answer.
3. Cluster and score options.
4. Pick next concrete experiment or draft.
