# Excalidraw — Compact

## Trigger

Use to create hand-drawn style diagrams: architecture, process, legal workflow, timelines, org charts and concept maps.

## Workflow

1. Clarify diagram type, audience and output path.
2. Draft nodes, groups, edges and labels before rendering.
3. Keep mobile/Feishu readability: fewer nodes, short labels.
4. Generate/edit Excalidraw artifact.
5. Verify file exists and preview/screenshot if needed.

## Output

Return absolute file path and brief diagram description.
