# Nano PDF — Compact

## Trigger

Use to edit PDF text, typos or titles via nano-pdf style workflow.

## Workflow

1. Verify PDF path and backup original.
2. Identify exact text/page to edit.
3. Apply edit to new output file unless overwrite authorized.
4. Verify output exists and sample page/text.
