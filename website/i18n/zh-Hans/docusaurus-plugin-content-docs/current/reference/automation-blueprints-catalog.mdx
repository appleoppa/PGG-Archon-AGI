# Automation Blueprints（自动化蓝图）

Automation Blueprints 是可直接运行的自动化模板。选择一个蓝图，填写少量字段，Hermes 就会把它调度为 cron 任务——不需要手写 cron 语法。

每个蓝图都可从**所有入口**使用：

- **Dashboard / 桌面 App** — 打开 Cron 页面，切换到 **Blueprints** 标签页，填写表单并点击 *Schedule it*。
- **CLI、TUI 和消息平台** — 输入 `/blueprint <name>`（例如 `/blueprint morning-brief`），Hermes 会逐项询问所需信息，然后创建调度任务。名称匹配是宽松的——前缀或近似拼写都可解析。高级用户也可以内联传值跳过提问：`/blueprint morning-brief time=08:00`。
- **桌面 App** — 在任意蓝图上点击 **Send to App**，会在输入框中预填该命令。

Blueprints 不会静默创建任何任务——创建前总会要求你确认。已创建的任务可随时通过 `/cron` 管理。

## 编写自己的蓝图

蓝图本质上是一个在 `SKILL.md` frontmatter 中带有 `metadata.hermes.blueprint` 块的 skill。槽位 schema 和发布方法见 [Creating Skills → Automation Blueprints](../developer-guide/creating-skills)。
